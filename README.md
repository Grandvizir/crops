# crops
Simple image crop and resize server

## Config
.env variables:
```
PORT=1337
BASE_SERVER=http://s3.amazonaws.com/
```
## Install dependencies
```sh
sudo apt-get install imagemagick graphicsmagick
```

## Run!
```
npm start
```
